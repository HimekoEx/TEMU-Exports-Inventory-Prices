# TEMU Exports Inventory Prices

是一个用于 TEMU 卖家中心的用户脚本, 它可以帮助卖家导出库存价格并屏蔽弹窗

## 功能特点

- 拦截并记录 TEMU 卖家中心的 商品列表 API 请求
- 在商品列表页面添加导出按钮, 将此次请求的 API 数据导出成 JSON 文件
- 屏蔽页面弹窗
